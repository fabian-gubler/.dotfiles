## Installation

```bash
# Clone this repository
git clone https://github.com/fabian-gubler/nixos-config

# Clone Dotfiles
git clone https://github.com/fabian-gubler/.dotfiles

# Build system
sudo nixos-rebuild switch

# Dotfiles Configuration
bash config/configure.sh
```
